Este proyecto consiste en el desarrollo de una página web pensada como una biblioteca digital interactiva, donde los usuarios puedan escribir, leer y crear sus propios libros de manera sencilla. La plataforma permitirá que cualquier persona pueda desarrollar historias, poemas u otros tipos de textos, organizándolos en capítulos con títulos personalizados.
El objetivo principal es brindar un espacio accesible y gratuito, en el que los usuarios no necesiten pagar ni tener conocimientos avanzados de tecnología para poder utilizarlo. Se busca que la interfaz sea intuitiva, permitiendo crear contenido fácilmente mediante opciones claras como agregar nuevos capítulos, editar textos y explorar escritos.
Además, la página estará pensada tanto para quienes desean escribir como para aquellos que solo quieren leer cuentos, historias o poemas. De esta manera, cualquier usuario podrá disfrutar del contenido disponible sin necesidad de crear sus propios textos, fomentando así el hábito de la lectura.
En conjunto, esta plataforma busca incentivar la creatividad, la escritura y la lectura, ofreciendo un entorno digital simple, gratuito y accesible para todos
